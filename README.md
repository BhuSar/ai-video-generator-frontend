# AI Video Generator

A full-stack AI-powered video generation application that converts natural language prompts into downloadable MP4 videos.

## Live Demo

https://ai-video-generator-dpen.onrender.com/

## Features

- Generate AI-powered cinematic scenes from text prompts
- Create multiple AI-generated frames using OpenAI
- Automatically compile frames into MP4 videos using FFmpeg
- Video preview player
- Download generated videos
- Dark mode interface
- Responsive design

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express
- OpenAI API
- FFmpeg

### Deployment
- Render
- GitHub

## How It Works

1. User enters a text prompt
2. OpenAI generates multiple cinematic frames
3. Frames are saved on the server
4. FFmpeg combines frames into a video
5. Video is served to the frontend
6. User can preview and download the result

## Example Prompt

"A cinematic drone flythrough over the Grand Canyon at sunrise, golden light illuminating the canyon walls, realistic clouds, smooth camera movement, ultra realistic, 4K, National Geographic documentary style"
